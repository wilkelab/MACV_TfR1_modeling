This is a directory containing the template structure used in the homology modeling.


Contents:
1. 3KAS_unrenumbered.pdb - This is the the 3KAS structure which only the protein atoms. It has the original numbering. 
2. 3KAS_AB.pdb - This is the ACTUAL template structure that is used in the modeling protocol. The numbering of this structure is 							changed. It begins with the first residue numbered 1 and then it is sequentially numbered. 
3. renumber_pdb.py - This is the python script that was used to renumber the PDB structure.
