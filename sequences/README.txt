This directory contains the sequences used in the homology modeling process.
Contents:

plasmid_nucleotide_sequences.fasta - A file containing the nucleotide sequences of the viral glycoproteins used in the experiments
translated_gpcs.fasta - A file containing the amino acid sequences of the viral glycoprotein sequences
tfr1_sequences.fasta - A file containing all of the transferrin receptor sequences that were used in the modeling.
